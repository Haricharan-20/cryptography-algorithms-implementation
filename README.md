# Cryptography Algorithms Implementation

## Cyber Security Internship Project

### Objective
The objective of this project is to understand and demonstrate fundamental cryptographic techniques used to protect data and secure communications.

### Algorithms Implemented

1. AES-256-GCM
   - Symmetric encryption algorithm
   - Used for secure encryption and decryption of data
   - Demonstrates authenticated encryption

2. RSA-2048
   - Asymmetric cryptographic algorithm
   - Uses a public key for encryption and a private key for decryption
   - Uses OAEP padding with SHA-256

3. SHA-256
   - Cryptographic hashing algorithm
   - Converts input data into a fixed-length hash
   - Demonstrates data integrity and one-way hashing

### Technologies Used
- Python
- Python cryptography library
- hashlib
- Termux (Android)

### How to Run

AES:
python aes.py

RSA:
python rsa_encryption.py

SHA-256:
python sha256_hash.py

### Results
All three cryptographic demonstrations were successfully implemented and tested. AES and RSA successfully encrypted and decrypted sample messages, while SHA-256 successfully generated a cryptographic hash.

### Learning Outcomes
Through this project, I learned the basic concepts of symmetric encryption, asymmetric encryption, hashing, encryption keys, public/private key cryptography, and secure data protection.
